To install:
- Go to [DFRobot_MAX3010](https://github.com/DFRobot/DFRobot_MAX30102?tab=readme-ov-file) library.
- Download the .ZIP file
- Open Arduino IDE
- Go to Sketch > Include Library > Add .ZIP library
